# NgxRouterModule

## Setup

Import the `NgxRouterModule` into your root module

```typescript
import { NgxRouterModule } from '@ngx-extensions/extensions';

@NgModule({
 imports: [NgxRouterModule.forRoot()]
})
export class AppModule {}
```
